# dumb-thermostat
Classic on/off house thermostat

Based on arduino pro-mini
Use of DS1820 temperature sensor.
classic 8 char LCD
Li-ion 18650 power
2 bouttons
And a "magic" bi-stable relay 
All wiring are define in the code
